# Code de conduite

## Notre engagement

Dans l'intérêt de favoriser un environnement ouvert et accueillant, nous nous engageons,  
en tant que contributeurs et responsables, à faire de la participation à ce projet une  
expérience exempte de harcèlement pour tout le monde, quel que soit l'âge, le handicap,  
l'origine, l'identité ou l'expression de genre, le niveau d'expérience, la nationalité,  
l'apparence, la religion ou l'orientation sexuelle.  

## Nos critères

Exemples de comportements positifs :
- faire preuve d'empathie et de bienveillance ;
- respecter les opinions et expériences différentes ;
- accepter les critiques constructives avec élégance ;
- se concentrer sur ce qui est le mieux pour la communauté.

Exemples de comportements inacceptables :
- propos ou images à caractère sexuel, avances non sollicitées ;
- trolling, insultes, attaques personnelles ou politiques ;
- harcèlement public ou privé ;
- publication d'informations privées d'autrui sans permission.

## Responsabilités

Les responsables du projet sont chargés de clarifier les critères de comportement  
acceptable et sont censés prendre des mesures correctives appropriées et équitables en  
réponse à tout comportement inacceptable.  

## Périmètre

Ce code s'applique dans tous les espaces du projet ainsi que lorsqu'une personne représente  
le projet ou sa communauté dans des espaces publics.  

## Application

Les cas de comportement abusif, harcelant ou autrement inacceptable peuvent être signalés
à **NDXDev@gmail.com**. Toutes les plaintes seront examinées et traiteront avec discrétion.

## Attribution

Ce code de conduite est adapté du [Contributor Covenant](https://www.contributor-covenant.org),  
version 2.1.
